# 💻 Projeto de Software - PUC Minas Coração Eucarístico 📚

Aqui estão organizados materiais, anotações, implementações e exercícios relacionados aos artigos acadêmicos estudados ao longo da matéria. O foco principal é utilizar o repositório como referência prática para compreensão dos conceitos teóricos apresentados em sala, facilitando a aplicação dos conteúdos em projetos reais.

📌 Este projeto tem finalidade exclusivamente acadêmica, servindo como material de estudo e apoio à disciplina de Projetos de Software.
